## Usage

To run this project locally, follow these steps:

1. Clone the repository:

   git clone https://github.com/juanycorn/React_Portfolio

2. Navigate to the project directory:

    cd React_Portfolio

3. Install dependencies:
    
    npm install

4. Run the project:

    npm start
